# feedback-soundfonts

Public host for the soundfont assets the feedback desktop app downloads
(kept out of the private app repos so the build and the in-app download
work without auth).

## Assets (`soundfonts-v1` release)
- **GeneralUser-GS.sf2** — bundled default. © S. Christian Collins, freely redistributable. See https://schristiancollins.com/generaluser.php
- **FluidR3_GM.sf2** — optional high-quality download. FluidR3 GM by Frank Wen, MIT-style licence.

These are standard General-MIDI soundfonts; no third-party game content.
